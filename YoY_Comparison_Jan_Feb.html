January: Fewer Users, More Engaged (Web)

Jan 2026 vs Jan 2025 · Weekday averages
📉 Web applicants fell -13.9% (931 → 802/day) — biggest volume drop across all metrics
📉 Total applicants fell -8.9% (2,896 → 2,637/day) — platform-wide softness
📈 Web apps/applicant rose +10.6% (5.11 → 5.65) — web users who did come applied MORE
➡️ Total apps/applicant flat +0.1% (4.82 → 4.82) — overall engagement held steady
📉 Total applications -9.1% (14,046 → 12,762/day) — purely volume-driven decline
🔑 Key Signal: Volume Down, Quality Up (Web)
Jan 2026 saw fewer premium users on web desktop, but those who did visit applied to 10.6% more jobs. This suggests the web channel retained its most engaged users — a potential quality-of-traffic improvement, or a seasonal effect where lower-intent users dropped off in Jan 2026.
📆 Worst DOW: Thursday (-23% web applicants)
Thursday drove the biggest single-day web applicant decline (970 → 745, -23%), yet its ratio improved from 4.99 to 5.70 (+14%). Friday also saw a large ratio jump (4.91 → 5.83, +19%). All 5 weekdays had lower volumes but higher web engagement ratios in Jan 2026.

February: More Applicants, Fewer Applications

Feb 2026 vs Feb 2025 · Weekday averages
➡️ Web applicants flat -0.1% (960 → 959/day) — essentially unchanged YoY
📈 Total applicants +3.9% (2,969 → 3,084/day) — more premium users overall
📉 Web applications -8.0% (5,324 → 4,899/day) — significant drop despite flat visitors
📉 Total applications -7.8% (15,228 → 14,037/day) — fewer applies despite more applicants
📉 Total apps/applicant -11.4% (5.13 → 4.55) — sharpest engagement decline
🔑 Key Signal: Engagement Depth Declined Structurally
Feb 2026 had virtually the same number of web applicants as Feb 2025, yet each applicant submitted 8% fewer web applications and 11.4% fewer total applications. This is not a traffic problem — premium users are visiting but applying to fewer jobs per visit.
📆 Worst DOW: Monday & Thursday
Monday web applications fell -12.7% (6,540 → 5,709) despite +1.8% more applicants, crushing the Monday ratio (6.29 → 5.35, -15%). Thursday web applications fell -15.2% (5,347 → 4,534). Wednesday was the only day with a positive applications trend (+1.9%).
